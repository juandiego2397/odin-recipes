# odin-recipes

Primer Proyecto de recetas 

Este es el primer proyecto realizado con el curso "The Odin Proyect", donde me encuentro realizando una pagina web que contiene recetas, donde se pudieron utilizar varios conceptos aprendidos en HTML. Los conceptos utilizados mas comunes fueron:

1. configuración del Git y Guthub para poder realizar el proyecto desde mi ordenador y luego por medio de comandos poder subir el proyecto de forma remota.
2. generar archivos HTML y directorios para poder comenzar a generar el Proyecto de recestas.
3. Conceptos basicos de como generar: titulos, listas, agregar enlaces, agregar imagenes y otros conceptos adicionales.
4. por ultimo generar la visualización de la pagina web

Nota: este primer proyecto no lleva estilo ya que por el momento se esta generando conceptos de HTML. Mas adelante se espera que despues de seguir con el curso y tomar la seccion de CSS se pueda pasar por este proyecto y agregarle estilo a la pagina web.
